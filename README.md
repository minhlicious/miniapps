# Mini Apps — Interactive Learning Tools

🌐 **Live Demo:** [https://miniappshowcase.netlify.app/](https://miniappshowcase.netlify.app/)


🌐 **Live Demo 2:** [https://minhlicious.github.io/miniapps/](https://minhlicious.github.io/miniapps/)

A curated collection of interactive mini apps for learning electronics, physics, vocabulary, and music. Built entirely with vanilla HTML, CSS, and JavaScript.

## 🚀 Apps Collection

### ⚡ Circuit Simulator
Interactive DC analysis lab — build circuits with resistors, LEDs, and a switch. Watch animated current flow in real time.
- **Features:** Canvas, Ohm's Law, Interactive

### 📖 Lexicon Flashcards
Elegant vocabulary trainer with 3D card flips, score tracking, and progress bar. Master advanced English words one card at a time.
- **Features:** Vocabulary, 3D Flip, Scoring

### 🔬 Physics Formula Solver
Step-by-step physics solver for Ohm's Law, Kinetic Energy, and Newton's Second Law. Enter known values and watch the solution unfold.
- **Features:** Physics, Step-by-Step, Formulas

### 🎹 Grand Piano
A two-octave interactive piano with touch and mouse support, realistic key animations, and Web Audio synthesis. Play it anywhere.
- **Features:** Web Audio, Touch, 2 Octaves

## 🛠️ Technology
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript

## 🚀 Deployment (GitHub Pages)

This project consists of static HTML, CSS, and JavaScript files and can be easily deployed to GitHub Pages.

1. **Initialize a Git repository** (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. **Create a GitHub repository** and push your code:
   ```bash
   git branch -M main
   git remote add origin <your-repo-url>
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to your repository **Settings** on GitHub.
   - Navigate to the **Pages** section on the left sidebar.
   - Under **Build and deployment**, select **Deploy from a branch**.
   - Choose the `main` branch and `/ (root)` folder, then click **Save**.
   
Your site will then be available at `https://<username>.github.io/<repository-name>/`.
